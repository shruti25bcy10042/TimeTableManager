# Timetable Manager for Students

A simple Python-based timetable manager to help students avoid class clashes.

## Features
- Add classes to your timetable
- Automatic clash detection
- View your weekly schedule
- Simple command-line interface

## Technologies Used
- Python 3
- Basic data structures (lists, dictionaries)

## How to Run
1. Clone this repository
2. Run the script:
```bash
python timetable.py
